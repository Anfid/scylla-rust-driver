# scylla-rust-driver
Async CQL driver for Rust, optimized for Scylla!

## Getting Started

You can run the [example](examples/basic.rs) program as follows:

```sh
SCYLLA_URI="127.0.0.1:9042" cargo run --example basic
```

## Features

* Asynchronous API
* Prepared statements
* Paging
* Compression (LZ4 and Snappy)
